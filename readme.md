> Compare an unsure array against a set of values to check if it&#39;s legit


## Install

```
$ npm install --save arr-u-legit
```


## Usage

```js
const arrULegit = require('arr-u-legit');

arrULegit('unicorns');
//=> 'unicorns & rainbows'
```
